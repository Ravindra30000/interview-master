**Format**: Screen recording with voiceover

## Production Tips

1. **Recording Quality**:

   - Use screen recording software (OBS, Loom, or QuickTime)
   - Record at 1080p minimum
   - Ensure clear audio

2. **Pacing**:

   - Keep transitions smooth
   - Don't rush through features
   - Pause briefly on key points

3. **Highlighting**:

   - Use cursor movements to guide attention
   - Zoom in on important metrics
   - Show actual data, not placeholders

4. **Voiceover**:

   - Speak clearly and confidently
   - Match tone to professional context
   - Practice script beforehand

5. **Editing**:
   - Add subtle transitions
   - Include text overlays for key stats
   - Ensure smooth flow between scenes
